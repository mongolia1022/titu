    DocCms2016 X1.0[音译:稻壳Cms]企业信息建站系统简介

+---------------------------------------------------+

    DocCms X — [ 音译：稻壳Cms ] 是一款于2013年12月21日正式发布，定位于为企业、站长、开发者、网络公司、VI策划设计公司、SEO推广营销公司、网站初学者等用户量身打造的一款全新企业建站、内容管理系统，服务于企业品牌信息化建设，也适应用个人、门户网站建设！

    DocCms X 前身基于ShlCms 4.2 [ 中文名：深喉咙Cms ] 演化而来，继承了前 ShlCms 用户操作界面清爽友好、后台使用简单便捷、模板模块儿开发制作快速灵活 三大优点！同时秉承原 ShlCms 永久免费开源的方针，继续为广大用户造福。

+---------------------------------------------------+

    稻壳Cms软件对环境运行的需求：

    1. 可用的 www 服务器，如 Apache、Zeus、IIS、Nginx 等  
    2. php 5.0 及以上 
    3. MySQL 5.0 及以上
    4. 服务器支持 urlrewrite（非必须）
    5. 服务器开启 gd 扩展
    6. 服务器开启 ob_start 缓存

+---------------------------------------------------+

    稻壳Cms软件的安装：

    非集成环境版本上传安装：
    1. 上传 doccms 目录中的所有文件到服务器虚拟主机根目录；
    2. 执行安装脚本程序：
       在浏览器中输入http://您的域名/setup/setup.php 运行 安装 程序，即访问 http://您的域名/setup/setup.php ；
    3. 参照页面提示，填写相应的主机信息，进行安装，直至安装完毕。

    集成环境版本本地安装测试(暂未推出)：
    1. 停止本地电脑原有相关php环境及mysql或相关sql服务；
    2. 双击下载到的exe文件进行解压，并按提示进行安装。
